Projeto de automação de interface de usuário (UI) utilizando o framework Cypress. 
O objetivo é implementar testes end-to-end para aplicações web, 
assegurando a confiabilidade e a estabilidade das funcionalidades críticas por meio de execuções automatizadas, 
integração contínua, boas práticas de organização de testes e validações consistentes do comportamento esperado.

UI automation project with Cypress. A project focused on automating user interface (UI) tests using the Cypress framework. 
The goal is to implement end-to-end tests for web applications, ensuring the reliability and stability of critical functionalities through automated test execution, 
continuous integration, best practices for test organization, and consistent validation of expected behavior.
